# 📊 Global Shipment Performance Dashboard

An interactive Power BI dashboard built to explore shipment, sales, and profitability data through business-focused visualizations and KPIs. This project demonstrates an end-to-end Power BI workflow, from preparing raw data to building an interactive dashboard for analysis and decision-making.

## 📌 Project Overview

This dashboard brings together shipment, product, and sales data into a single interactive report. It focuses on presenting key business metrics in a clean and easy-to-understand format while allowing users to explore trends, product performance, geographic distribution, and salesperson insights.

---

## ✨ Features

- 📈 KPI cards for Total Amount, Total Profit, Shipment Count, Total Boxes, and Profit %
- 📊 Year-over-Year comparison of Amount and Boxes
- 🚚 Shipment Distribution analysis
- 🌍 Sales distribution by geography
- 📦 Top-performing products
- 👥 Salesperson performance comparison
- 🎨 Interactive and business-friendly dashboard layout
